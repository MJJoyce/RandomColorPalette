RandomColorPalette.js

Sometimes you need to generate a random number of colors that are still distinct enough for pleasant viewing. That's when you grab RandomColorPalette.js!! It doesn't promise that it'll generate a perfect color palette. In fact, sometimes there will be colors that are fairly close together. However, it will do a reasonably good job without trying too hard and without knowing how many colors you need in advance.

Let's grab a few random colors and see what we end up with. 

var colorGen = new ColorPaletteGenerator();
var newColor = colorGen.getNextRGBHexColor();

Each call to getNextRGBHexColor() gives us a random color. Here's an example of what doing this a few time might give us!

We can also be a bit more picky about how we get the colors that we want. We can set the seed value for the hue so we get the same series of colors each time. Since we're dealing with colors in HSV we can also play with the saturation and value to change the palette.

Here's an example that gives us a pastel look. 

colorGen.setHue(0.9927902359049767).setSaturation(0.5).setValue(0.95);
